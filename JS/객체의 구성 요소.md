# 객체의 구성 요소

객체 안에 있는 함수 형태의 속성은 **메서드(Method)** 라고 부른다.

객체는 일반적으로 **속성(Property)**과 **메서드(Method)**로 구성된다.

### 속성(Property)

객체가 가지고 있는 데이터나 상태를 나타내는 이름-값 쌍(name-value pair)이다. 값은 문자열, 숫자, 배열 등 다양한 자료형이 될 수 있다.

### 메서드(Method)

객체가 수행할 수 있는 동작이나 행위를 나타내는 함수이다. 이 함수는 객체에 속한 데이터를 조작하거나 특정 연산을 수행한다.

예를 들어, JavaScript에서 person이라는 객체가 있을 때:

```jsx
var person = {
  name: "Alice", // 속성 (Property)
  greet: function () {
    // 메서드 (Method)
    console.log("Hello, " + this.name + "!");
  },
};

person.greet(); // 메서드 호출
```

이 코드에서 name 은 속성, greet은 메서드이다.
